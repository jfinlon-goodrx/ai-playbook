# Task: UI Slice From Figma for React

## Prompt

```text
@src/components/[ExistingComponent].tsx
@src/pages/[ExistingPage].tsx
@src/[theme or design-system file]
@src/[existing UI test file]

Use this Figma handoff to implement a React UI slice.

Design context:
- Screen or component: [name]
- Figma summary: [layout, sections, variants, tokens, states]
- Responsive behavior: [requirements]
- Accessibility notes: [requirements]
- Data dependencies: [API calls and state sources]

Requirements:
- TypeScript with strict typing
- reuse existing component and hook patterns
- match the current UI library and theme
- include loading, empty, error, and validation states
- add tests with React Testing Library or the repo's existing UI test stack
- call out any design ambiguities before implementation
```
