# Backend Workflows

This folder contains backend delivery workflows organized as:

- one shared workflow file
- one `.dotnet` companion
- one `.python` companion
- one `.go` companion

Use the shared file first to understand the task shape, then open the language companion that matches your stack.

## How To Use This Folder

1. Start with the shared workflow.
2. Open the companion that matches your stack.
3. Point the assistant at real repo files before asking it to generate changes.
4. Ask for assumptions, risks, and tests before asking for implementation.
