## First Three Wins

1. Improve one story template to emphasize intent, constraints, and rollout needs.
2. Add explicit in-scope, out-of-scope, and success-measure fields to stories.
3. Align review expectations before implementation starts.

## Working Rules

- write for clarity of intent, not implementation micromanagement
- include rollout and observability expectations for meaningful changes
- keep acceptance criteria testable
- classify risk before development begins
