# DevOps and Platform Starter

## Goal

Help platform and DevOps engineers use AI for safer pipelines, environment workflows, incident response, and operational documentation.

## First Three Wins

1. Draft or review one service pipeline.
2. Use AI to triage one real pipeline failure.
3. Improve one deployment or rollback runbook.

## Working Rules

- treat secrets, permissions, and environment references as high-risk
- use placeholders for environment-specific values
- require rollback thinking for non-trivial changes
- prefer explicit deployment stages over magical YAML
