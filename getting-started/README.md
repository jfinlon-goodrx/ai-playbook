# Getting Started

This section helps newer team members go from zero to useful with AI-assisted work.

## Recommended Order

1. `01-installing-cursor.md`
2. `02-your-first-prompt.md`
3. `03-project-context.md`
4. `04-when-to-use-ai.md`
5. `05-first-week-with-ai.md`

## Best First Wins

- Understand how project context changes output quality: `03-project-context.md`
- Learn when to trust AI and when to slow down: `04-when-to-use-ai.md`
- Build confidence through low-risk practice: `05-first-week-with-ai.md`

## Rule Of Thumb

The fastest path to confidence is:

1. use real repo context,
2. start with reviewable tasks,
3. verify the output,
4. save what worked for reuse.
