# 0 to 60 AI

This folder is a curated subset of the larger AI education curriculum, trimmed for teams that need practical adoption help more than a full book-length course.

It is intentionally biased toward:

- prompt quality and working with AI well,
- tool use, skills, and MCP,
- role-based guidance for developers, QA, DevOps, and engineering leaders,
- and the production disciplines that become mandatory once AI matters.

## When To Use This Folder

Use this section when:

- a team wants a shared AI vocabulary without reading the entire source curriculum,
- a manager needs role-specific onboarding material,
- QA or DevOps needs a faster path into AI-assisted work,
- or you need a practical bridge between experimentation and production discipline.

## What This Is Not

This is not the full `Ai-Education` project. It is the high-value subset that best supports a dotnet/Azure delivery organization using the rest of this playbook.
